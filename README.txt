# RPG Game

This project is a console-based role-playing game (RPG) where the player battles against various enemies using chosen weapons. The game continues in turns until the player is defeated.

## Features

- **Character Customization**: Choose your weapon at the beginning of the game.
- **Turn-Based Combat**: Fight against enemies in a turn-based system.
- **Dynamic Enemy Generation**: Enemies are generated with random attributes.


## Installation

To run this game, you'll need to install [.NET](https://dotnet.microsoft.com/download) on your machine.


Game Mechanics
Player: Initiates with chosen weapon and fixed health.
Enemy: Generated with random health and a randomly assigned weapon.
Combat: Turns alternate between the player and the enemy. Each attack is processed asynchronously.
